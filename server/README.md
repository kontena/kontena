# Kontena Server

## Features

* Access control
* Orchestrates multiple Docker grids (clusters)
* Container logs
* Container metrics
* Audit log
* Service abstraction
* Scheduler
* Realtime (websocket) channel to Kontena Agents

## Development

### Get started

Start app
```sh
$ docker-compose up
```

Build:

```sh
$ docker build -t kontena/server .
```
